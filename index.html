💾 Azure Storage (ストレージサービス) 💾
🗄️ 無制限ストレージサービス 🗄️
データを安全に、高速に、経済的に保存！99.999999999%（イレブンナイン）の堅牢性を実現！

ストレージの種類：
  • Blob Storage - オブジェクトストレージ（画像、動画、バックアップに最適）
  • File Storage - 共有ファイルシステム（SMB/NFS対応）
  • Queue Storage - メッセージキュー
  • Table Storage - NoSQLデータベース
  • Disk Storage - 高性能SSD/HDD

🌐 Azure App Service (Webアプリホスティング) 🌐
🚀 サーバーレス Web アプリ 🚀
サーバー管理不要！コードをデプロイするだけで本格的なWebアプリケーションが稼働！

対応言語・フレームワーク：
  • .NET - ASP.NET Core、ASP.NET Framework
  • Java - Spring Boot、Tomcat
  • Node.js - Express.js、React、Vue.js
  • Python - Django、Flask
  • PHP - Laravel、WordPress

⚡ Azure Functions (サーバーレス関数) ⚡
🔧 イベント駆動コンピューティング 🔧
実行した分だけ課金！サーバー管理一切不要の超効率的な関数実行環境！

トリガーの種類：
  • HTTP トリガー - REST API として利用
  • Timer トリガー - 定期実行（cron式）
  • Blob トリガー - ファイルアップロード時に実行

  • Cosmos DB - データベース変更で実行
実行 100万回まで無料！
